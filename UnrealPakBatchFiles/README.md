### UnrealPak Batch Files
```
A collection of batch files for working with pak files.

extract - extracts pak
create - creates pak file from compiled files (obtained from extract)
compress - compress pak file

How-To
copy UnrealPak.exe from the devkit files, copy pak/folder to the
directory and drag and drop onto the batch file.

extract/compress support .pak file
create supports folders 
```

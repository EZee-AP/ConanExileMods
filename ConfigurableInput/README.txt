Configurable Key

For the key to be configurable in the editor you must add a new action mapping in "Project Settings->Input->Action Mappings" 
For the key to be configurable in game you must add this line to the "DefaultInput.ini" found in "Game Install Dir\ConanSandbox\Config\DefaultInput.ini"

+ActionMappings=(ActionName="CustomKey",Key=ThumbMouseButton2,bShift=False,bCtrl=False,bAlt=False,bCmd=False)
